📌 Project Title

AI-Based Legal Document Summarizer & Law Advisor using machine learning 

🚀 Description

This project is an AI-powered web application that automatically summarizes complex legal documents and provides relevant insights using Machine Learning and Natural Language Processing (NLP). It helps users understand lengthy legal content quickly and efficiently.

🛠️ Tech Stack
Programming Language: Python
Framework: Flask
Machine Learning: BERT, Naïve Bayes, SVM
NLP Techniques: TF-IDF, Named Entity Recognition (NER)
Database: MySQL / SQLite
Libraries: NLTK, Scikit-learn, Transformers

✨ Features
Upload legal documents (PDF, DOCX, TXT)
Automatic text summarization (extractive & abstractive)
Document classification (contracts, judgments, notices)
Key information extraction (entities like names, dates, clauses)
Simple and user-friendly interface
📂 Project Structure
app.py              → Main Flask application  
templates/          → HTML files  
static/             → CSS, JS, images  
models/             → ML models  
uploads/            → User uploaded files  
⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/legal-summarizer.git
cd legal-summarizer
2. Install dependencies
pip install -r requirements.txt
3. Run the application
python app.py
🌐 Usage
Upload a legal document
Click on “Generate Summary”
View summarized content and document type
Get key insights and extracted information
